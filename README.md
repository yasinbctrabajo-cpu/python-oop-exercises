# Programación Orientada a Objetos en Python

## Descripción

Repositorio que contiene ejercicios prácticos centrados en la Programación Orientada a Objetos (POO) utilizando Python.

Estos ejercicios permiten comprender cómo modelar objetos y estructuras complejas mediante clases y métodos.

## Tecnologías utilizadas

- Python 3
- Jupyter Notebook

## Conceptos trabajados

- Clases
- Objetos
- Atributos
- Métodos
- Constructor (__init__)
- Encapsulación
- Representación de objetos (__str__)

## Objetivo

Desarrollar programas utilizando los principios básicos de la programación orientada a objetos para crear soluciones más escalables y organizadas.
